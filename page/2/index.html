hdu5372(树状数组)

发表于 2015-08-12   |   分类于 acm   |  

题目链接

####题意：

给定$n(n \le 2 \cdot 10^5)$次操作，每次操作分两种：

  1. $0\, x:$在数轴上加入一条线段$[x,x+i]$，其中$i$代表该线段是第$i$次加入的。在加入该线段之前，问：当前数轴上有多少条线段被该线段包含。
  2. $1\, x:$删除第$x$次加入的线段。

####题解：

考虑原问题的逆问题，即：有多少条线段不被当前线段$[l, r]$包含。这时有两种情况：

  1. 线段之间没有交点，即线段的右端点小于$l$或者线段的左端点大于$r$，这种情况可以用树状数组来解决。
  2. 线段之间有交点，这时候如果线段$[l1,r1]$不被包含，则一定要满足：$l1 < l \le r1$或者$l1 \le r < r1$（新加入的线段比之前加入的线段长）。这种情况也可以用树状数组来解决。

时间复杂度：$O(nlogn)$

hdu5356

发表于 2015-08-11   |   分类于 acm   |  

题目链接

####题意：

给定一棵有$n(n \le 2000)$个节点的树，刚开始第$i$个节点有一种颜色$i$，所有的边都没有颜色。现在要对这棵树做$m(m \le 10^9)$次操作，每次操作如下：

  1. 选定一个节点$v$，以及$v$中的一种颜色$i$。
  2. 选定另一个节点$u$。
  3. 将$u$到$v$路径上所有点和边都染上颜色$i$，颜色之间不会覆盖。

第$i$种颜色有一个权值$ci$，设$li$为$m$次操作后含有颜色$i$的边的长度总和。问$\sum_{i=1}^n ci \cdot li$的最大值是多少。

####题解：

首先，颜色之间是互不影响的，且每种颜色最多做$n$次操作，之后的操作不影响答案。设$f[i][j]$代表对第$i$种颜色做$j$次操作，最多能获得的权值。

计算$f[i][j]$：

将节点$i$当做根，问题就转化为，选择$j$个节点，将根节点到这$j$个节点的路径上所有边都打上标记，被标记的边的长度总和最大是多少？

对于被选择的$j$个节点，肯定是选择叶节点比较优。到这里，有一种贪心的方法可以解决这个问题：

选择一个叶节点$v$，满足从根节点到$v$的路径最长，路径长度用于更新$f$数组。然后把节点$v$到根节点路径上的点和边全删掉(除了根节点)，此时可能会产生一些树，且根上有一条只有一个端点的边，我们直接把这条边接到原来的根节点。重复此操作，直到只剩下根节点为止。

这样我们就能在$O(n)$时间内处理出$f[i][j]$，然后就很容易算出答案了。

ural1918(容斥)

发表于 2015-08-10   |   分类于 acm   |  

题目链接

####题意：

给定一个整数$n(n \le 5000)$，问：有多少种长度为$n$的序列$a$合法。合法的序列满足：

  1. $1 \le ai \le n$
  2. 将序列对应一个有$n$个节点的图，其中$i$到$i-1$($1 < i \le n$)有一条有向边，$i$到$ai$有一条有向边。节点$1$到节点$n$有一条路径。

####题解：

有题意可知，如果从节点$1$可以到节点$j$，那么从节点$1$可以到节点$k(k \le j)$。设：

$f[i]$ 代表只考虑前$i$个节点，且$1$号节点可以到达$i$号节点的方案数。
$g[i]$ 代表只考虑前$i$个节点的方案数。

$g[i] = i ^ i$

$g[i] = \sum_{j=1}^i f[j] \cdot i^{i-j}$

化简得到：

$f[1] = 1$

$f[i] = i^i - \sum_{j=1}^{i-1} f[j] \cdot i^{i-j}$

然后就可以$O(n^2)$随便做了。

uvalive6923(dp)

发表于 2015-08-09   |   分类于 acm   |  

题目链接

####题意：

问，有多少个$n(3 \le n \le 2000)$的排列$a$满足：$(ai, i)$是相邻的$i$的个数不小于$m(m \le n)$，$i$和$i+1$以及$i-1$是相邻的，$1$和$n$是相邻的。

####题解：

$f[i][j][k1][k2][l1][l2]$ 代表已经处理了前$i$个数字，有$j$个数字满足$(ai,i)$是相邻的，只把这$j$个数字放入排列，$k1$是第$n$个位置是否为空，$k2$是第$1$个位置是否为空，$l1$是第$i$个位置是否为空，$l2$是第$i + 1$个位置是否为空，在这些条件下的方案数。

关于转移，只要枚举第$i + 1$个数是否放入排列以及放到$i-1$、$i$还是$i+1$，这样就可以在$O(1)$时间内完成转移。

令$dp[j] = \sum f[n][j][k1][k2][l1][l2] \times (n - j)! (k1和k2至少一个为空，l1和l2至少一个为空)$

$g[j] = dp[j] - \sum_{i = j + 1}^n g[i] \times \binom{i}{j}$。

$ans = \sum_{i = m}^n g[i]$

$dp[j]$代表至少有$j$个数字满足条件，但是会有重复。

$g[j]$相对于对$dp[j]$进行了一次容斥。

时间复杂度：$O(n^2)$

####总结：
$dp[j]$是有可能大于$n!$的，之前一直以为$dp[j]$怎么也不可能大于$n!$。训练的时候看了一个多小时都没发现错，赛后才发现原来根本没有错。

uvalive6132(dp)

发表于 2015-08-08   |   分类于 acm   |  

题目链接

####题意：

有$B(2 \le B \le 100)$个盒子，第$i$个有$a[i]$个石子。现在有两个人玩游戏，每次都会重复以下操作：

  1. 第一个人对每个盒子确定一个$bi$。
  2. 第二个人有两种选择：一种选择是令$c[i] = bi$，另 一种选择是令$c[i] = a[i]-bi$。
  3. 令 $a[i] = ci-1$，$a[1] = 0$。

如果在某个时刻$a[B]$不等于0，则第一个人获胜。问在游戏开始前，有多少种不同的$a$，使得$\sum_{i=1}^B a[i] = S$，且第一个人不会获胜（假设两人都足够聪明）。

####题解：

给每个游戏的状态$f$定义一个权值为$val[f]$，设$val[f] = \sum_{i=1}^B \frac{a[i]}{2^{B-i}}$。如果$val[f] < 1$，第一个人必输，否则第一个人必胜。

证明：游戏的操作可以等价于，第一个人选择一个权值$g$，然后第二个人选择权值$2g$或者$2(val[f]-g)$作为下一轮游戏的权值。如果$val[f] < 1$，则$min(2g, 2(val[f] -g)) < 1$，那么第$B$个盒子永远也不会有石子。如果$val[f] >= 1$，那么只需要令$g = \frac{1}{2}$(容易证明，如果$a[B] = 0$，$g$一定能取到$\frac{1}{2}$)就必然会有一个时刻使得$a[B] > 0$。

设$dp[i][j][k]$代表已经考虑了前$i$个盒子且放入了$j$个石子，当前局面的权值$val[f]$满足$\frac{k}{2^{B-i}} \le val[f] < \frac{k + 1}{2^{B-i}}$的方案数。

$dp[i][j][k] -> dp[i][j + 1][k + 1]$

$dp[i][j][k] -> dp[i + 1][j][k / 2]$

$dp[B][S][0]$就是答案。

时间复杂度：$O(BS^2)$

hdu2485(网络流)

发表于 2015-08-07   |   分类于 acm   |  

题目链接

####题意：

给一个有$n(1 \le n \le 50)$个节点$m(1 \le m \le 4000)$条边的图，问至少删除多少点(不能删除$1$号点和$n$号点)使得从$1$号点到$n$号点不存在小于等于$K(1 \le K \le 1000)$的路径。

####题解：

将所有从$1$号点到$n$号点长度小于等于$K$的路径都加入到新图$G$。原问题转化为：删除最少的点，使得在$G$中不存在从节点$1$到节点$n$的路径。如果题目要求的是删除最少的边，那么我们只需要对$G$做一遍最小割就行了。对于删除最少的点，只要把每个点拆成两个，中间连一条流量为$1$的边，把$G$中的边流量设置为无穷大，然后做一边最小割就行了。

至于如何得到$G$，只需要对原图做一遍$floyed$，把满足$dis[1][x] + 1 + dis[y][n] \le K$的边$(x,y)$加入$G$即可。

hdu5343(后缀自动机)

发表于 2015-08-06   |   分类于 acm   |  

题目链接

####题意：

给定两个字符串$S1$和$S2$，设$x$为$S1$的子串，$y$为$S2$的子串，问有多少种不同的$x+y$。

####题解：

设$F[ch]$：满足$S$是$S1$的子串且$S + ch$ 不是$S1$的子串的不同$S$的数量。

设$G[ch]$：满足$S$是$S2$的子串且S的首字母是$ch$的不同$S$的数量。

最后答案为：$\sum_{i=a}^{z} F[i] * G[i]$（空串要特殊处理一下）。

证明：对于一个$x+y$，假设它被统计了多次，则有两种情况：

  1. 存在一个 $x1$ 满足 $x1 + ch = x$，存在一个$y1$满足$y1 = ch + y$，这样使得$x1+y1 = x + y$。但是如果$x1 + ch = x$，那么$x1$就不会被加入$F[ch]$，而$y1$是存在$G[ch]$里的，所以这种情况不会发生。

  2. 存在一个$x1$满足$x1 = x + ch$ ，存在一个$y1$满足$y1 + ch = y$，这样使得$x1+y1 = x + y$。但是如果$x1 = x + ch$，那么$x$就不会被加入$F[ch]$，而$y$是存在$G[ch]$里的，所以这种情况不会发生。

所以$x+y$不会被统计多次，利用类似的方法可以证明出每个$x+y$都会被统计到。

求$F[ch]$的方法：

在$SAM$中，从起点到节点$v$对应了一些子串，假设$trans(v,ch) = null$，那么这些子串满足：加一个字符$ch$就不是$S1$的子串了。所以$v$对应的这些子串都可以统计到$F[ch]$里面。

时间复杂度：$O(n)$

求$G[ch]$的方法：

建立$SAM$，设$dp[i]$代表从第$i$个节点出发能访问的子串个数。则$G[ch] = dp[trans(init,ch)]$。

时间复杂度：$O(n)$

所以总时间复杂度：$O(n)$

hdu5352(网络流)

发表于 2015-08-05   |   分类于 acm   |  

题目链接

####题意：

有一个含有$n$个节点的图，刚开始的时候$n$个点都是被破坏的，且没有边。现在有$m$个操作，操作分为三种类型：

$1 \,x:$可以修复与$x$处于同一连通块的点，但是最多只能修复$K$个点。

$2 \, x \, y:$加入边$(x,y)$。

$3 \, p \, x1 \, y1 …xp \, yp:$删掉边$(x1,y1)(x2,y2)…(xp,yp)$。

问最多能修复多少城市，并输出字典序最小的方案。
方案：$(a1,a2,…,at)$，一共有$t$个$1$操作，$ai$代表第$i$个$1$操作修复的节点数量。

####题解：

对于第$i$个$1$操作建一个点$bi$并暴力处理出可以修复的节点集合，然后从$bi$到集合中的节点各连一条流量为$1$的边。源点向$bi$连一条流量为$K$的边，每个原图中的节点向汇点连一条流量为$1$的边。在新图中跑一边最大流就能得到答案。

现在问题是：如何找字典序最小的方案。这里有一个性质，如果$at$最多可以为$x$，那么在字典序最小的方案中满足$at = x$。

证明：假设字典序最小的方案中满足$at=y$，那么一定存在一个节点$j$($j$能被最后一个$1$修复)被第$i$个$1$操作修复了，我们只需要让最后一个$1$修复节点$j$就可以让字典序变小，所以假设不成立。

做法：删去源点出发的所有边，只加入$(S,bt)$这条边，跑一遍最大流，这样就得到了$at$。然后删掉$(S,bt)$，再加入$(S,bt-1)$，跑一遍最大流得到$at-1$。如此做$t$次就能解决问题。

codeforces14D(dp)

发表于 2015-08-04   |   分类于 acm   |  

题目链接

####题意：

给定一棵树，在树上找出两条点不相交的路径，并最大化路径长度的乘积（路径长度即路径上的边数）。

####题解：

枚举每条边，通过删除这条边使得原树变成两棵新树，用两棵新树直径的乘积更新答案，时间复杂度：$O(n^2)$。由于这道题的$n \le 200$，所以上述做法已经能够解决这道题了。其实我们只要稍做优化就可以在$O(n)$时间内解决这个问题。

我们在做树的换根操作时，假设将根从节点$x$换到节点$y$，我们需要计算出不考虑子树$y$时节点$x$的$dp$值，而原来节点$y$的$dp$值本来就不包括节点$x$，这和删边起到了相同的作用，所以我们可以在$O(n)$时间内通过换根来解决这个问题。

时间复杂度：$O(n)$

uvalive7011(数学)

发表于 2015-08-03   |   分类于 acm   |  

题目链接

####题意：

给定一个长度为$n(1 \le n \le 15)$的非负整数序列$S$和一个非负整数$a(0 \le a \le 10000)$，现在要求一个序列$T$，使得它在满足$0 \le T[i+1] - T[i] \le a$的情况下最小化$\sum_{i=1}^n (S[i] - T[i])^2$。

额外条件：设$f(k,x)$代表只考虑前$k$个元素，且第$k$个元素为$x$的最小代价。对于任意$k(1 \le k \le n)$，$f(k,x)$是单谷的。

####题解：

$f(1,x) = (S[1] - x)^2$，

$f(k,x) = (S[k] - x)^2 + min(f(k-1,y)) ( 2 \le k \le n 且x - a \le y \le x)$

设 $pos[k]$ 为使得$f(k,x)$最小化的$x$。由于$f(k - 1,x) $是单谷的，所以：

当$pos[k-1] \le x \le pos[k-1]+a$时，

$f(k,x)=(S[k]-x)^2 + f(k-1, pos[k - 1])$

当$pos[k-1] + a \le x$时，

$f(k,x) = (S[k]-x)^2 + f(k-1, x - a)$

当$x \le pos[k-1] $时，

$f(k,x) = (S[k] - x)^2 + f(k-1,x)$

由上述式子可以得出：$f(k,x)$在$f(k-1,x)$的$pos[k-1]$处将函数分为左右两部分，并且将右边部分向右平移$a$单位长度，然后中间部分是$y = f(k-1,pos[k-1])$(先不考虑$(S[k]-x)^2$部分，这部分相当于是一个常数，可以最后统一加)。显然$f(k,x)$是一个分段函数，且段数最多比$f(k-1,x)$多$2$。我们只需要把$f(k,x)$每一段存下来，然后暴力维护即可。

$f(k,x)$最多有$O(2n)$段，一共维护$n$次，所以时间复杂度为：$O(n^2)$
