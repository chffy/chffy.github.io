uvalive6923(dp)

发表于 2015-08-09   |   分类于 acm   |  

题目链接

####题意：

问，有多少个$n(3 \le n \le 2000)$的排列$a$满足：$(ai, i)$是相邻的$i$的个数不小于$m(m \le n)$，$i$和$i+1$以及$i-1$是相邻的，$1$和$n$是相邻的。

####题解：

$f[i][j][k1][k2][l1][l2]$ 代表已经处理了前$i$个数字，有$j$个数字满足$(ai,i)$是相邻的，只把这$j$个数字放入排列，$k1$是第$n$个位置是否为空，$k2$是第$1$个位置是否为空，$l1$是第$i$个位置是否为空，$l2$是第$i + 1$个位置是否为空，在这些条件下的方案数。

关于转移，只要枚举第$i + 1$个数是否放入排列以及放到$i-1$、$i$还是$i+1$，这样就可以在$O(1)$时间内完成转移。

令$dp[j] = \sum f[n][j][k1][k2][l1][l2] \times (n - j)! (k1和k2至少一个为空，l1和l2至少一个为空)$

$g[j] = dp[j] - \sum_{i = j + 1}^n g[i] \times \binom{i}{j}$。

$ans = \sum_{i = m}^n g[i]$

$dp[j]$代表至少有$j$个数字满足条件，但是会有重复。

$g[j]$相对于对$dp[j]$进行了一次容斥。

时间复杂度：$O(n^2)$

####总结：
$dp[j]$是有可能大于$n!$的，之前一直以为$dp[j]$怎么也不可能大于$n!$。训练的时候看了一个多小时都没发现错，赛后才发现原来根本没有错。
