uva6840(dp)

发表于 2015-08-16   |   分类于 acm   |  

题目链接

####题意：

有一个栈，栈中有$n(n \le 150)$个糖果，每个糖果有两种属性$ai$和$bi$($\sum_{i = 1} ^ {n} bi \le 150$)。现在有两个人玩游戏，两人轮流操作，每次有两种操作：

  1. 跳过这次操作，但是总能量减少$1$。(两人刚开始的能量分别为$A,B(A,B \le 10^9)$，且任意时刻能量不能小于$0$)
  2. 取走栈顶的糖果，获得$bi$分，且能量增加$ai$。

游戏一直进行到栈中没有糖果为止，问：如果两人都足够聪明，他们各获得多少分。

####题解：

注意一个性质，如果先手拿了栈顶的糖果会导致结果不优，那么后手在最优策略下也一定会避免拿栈顶的糖果，最终的结果就应该是两人一直跳过操作，直到某个人的能量不足为止。

显然，如果先手不想要栈顶的糖果，那么先手的能量就至少比后手多$1$。如果先手想要栈顶的糖果，那么先手不管剩多少能量都能取得栈顶糖果。可见，两人能量具体的值并不重要，重要的只是能量的差值。

设$f[i][j]$代表：只考虑第$i$个糖果到第$n$个糖果的子游戏，先手获得了$j$分至少需要的先手与后手能量的差值。

状态转移：

  1. 先手取栈顶的糖果。设 $S$为 $\sum_{j = i}^n bj$，则当前的后手为下一轮的先手，且能获得的权值$val$应该小于等于$S - j$。对于所有的$k(k > S - j)$，必须满足 $-(f[i][j] + ai) < f[i + 1][k]$，所以$f[i][j] = max(-f[i + 1][k] - ai + 1)$。
  2. 先手不取栈顶的糖果。那么当前的先手在下一轮还是先手，且能获得的权值$val$应该大于等于$j$。存在一个$k(k >= j)$满足$f[i][j] - 1 - ai \geq f[i + 1][k]$，以及$f[i][j] \geq 1$，所以$f[i][j] =max( min(f[i + 1][k] + 1 + ai), 1)$。

最后的答案：最大的满足$f[1][j] \le A - B$的$j$。

时间复杂度：$O(n \cdot {(\sum_{i = 1}^ n bi)}^2)$

####总结：

以前做博弈，基本都是把能量当作状态，而这道题却把分值当作状态，算是开了眼界。
