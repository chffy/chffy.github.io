uvalive7011(数学)

发表于 2015-08-03   |   分类于 acm   |  

题目链接

####题意：

给定一个长度为$n(1 \le n \le 15)$的非负整数序列$S$和一个非负整数$a(0 \le a \le 10000)$，现在要求一个序列$T$，使得它在满足$0 \le T[i+1] - T[i] \le a$的情况下最小化$\sum_{i=1}^n (S[i] - T[i])^2$。

额外条件：设$f(k,x)$代表只考虑前$k$个元素，且第$k$个元素为$x$的最小代价。对于任意$k(1 \le k \le n)$，$f(k,x)$是单谷的。

####题解：

$f(1,x) = (S[1] - x)^2$，

$f(k,x) = (S[k] - x)^2 + min(f(k-1,y)) ( 2 \le k \le n 且x - a \le y \le x)$

设 $pos[k]$ 为使得$f(k,x)$最小化的$x$。由于$f(k - 1,x) $是单谷的，所以：

当$pos[k-1] \le x \le pos[k-1]+a$时，

$f(k,x)=(S[k]-x)^2 + f(k-1, pos[k - 1])$

当$pos[k-1] + a \le x$时，

$f(k,x) = (S[k]-x)^2 + f(k-1, x - a)$

当$x \le pos[k-1] $时，

$f(k,x) = (S[k] - x)^2 + f(k-1,x)$

由上述式子可以得出：$f(k,x)$在$f(k-1,x)$的$pos[k-1]$处将函数分为左右两部分，并且将右边部分向右平移$a$单位长度，然后中间部分是$y = f(k-1,pos[k-1])$(先不考虑$(S[k]-x)^2$部分，这部分相当于是一个常数，可以最后统一加)。显然$f(k,x)$是一个分段函数，且段数最多比$f(k-1,x)$多$2$。我们只需要把$f(k,x)$每一段存下来，然后暴力维护即可。

$f(k,x)$最多有$O(2n)$段，一共维护$n$次，所以时间复杂度为：$O(n^2)$
