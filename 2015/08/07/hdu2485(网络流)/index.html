hdu2485(网络流)

发表于 2015-08-07   |   分类于 acm   |  

题目链接

####题意：

给一个有$n(1 \le n \le 50)$个节点$m(1 \le m \le 4000)$条边的图，问至少删除多少点(不能删除$1$号点和$n$号点)使得从$1$号点到$n$号点不存在小于等于$K(1 \le K \le 1000)$的路径。

####题解：

将所有从$1$号点到$n$号点长度小于等于$K$的路径都加入到新图$G$。原问题转化为：删除最少的点，使得在$G$中不存在从节点$1$到节点$n$的路径。如果题目要求的是删除最少的边，那么我们只需要对$G$做一遍最小割就行了。对于删除最少的点，只要把每个点拆成两个，中间连一条流量为$1$的边，把$G$中的边流量设置为无穷大，然后做一边最小割就行了。

至于如何得到$G$，只需要对原图做一遍$floyed$，把满足$dis[1][x] + 1 + dis[y][n] \le K$的边$(x,y)$加入$G$即可。
